# Notas de Configuración – Betaflight

## Puertos (Ports)
- UART1 → Serial RX (ELRS)
- UART5 → VTX (IRC Tramp)

## Configuración principal
- Max arm angle: 180°
- Nombre del dron: tu_nick / tinywhoop

## Modos
- ARM → AUX1
- ANGLE → AUX2
- BEEPER opcional

## OSD (Analógico NTSC)
Incluye:
- Voltaje
- Tiempo de vuelo
- RSSI
- Throttle
- Indicador de batería
- Modo de vuelo
- Logo personalizado
