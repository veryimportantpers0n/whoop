The Build

Este TinyWhoop ha sido diseñado para dominar espacios cerrados con la potencia de una configuración 2S. Utilizando el frame Meteor 75 Pro, logramos una agilidad increíble sin sacrificar durabilidad.

Equipado con ELRS para un enlace de radio sólido como una roca y un sistema de video analógico optimizado para penetración en entornos urbanos densos.
